# hydration-and-calorie-tracker

A simple desktop GUI application for tracking daily water intake and calorie
consumption.

## Overview

This project is a semester assignment focused on building a graphical database
application. It allows users to monitor hydration and calorie intake, set daily
goals, and review their progress over time.

## Features

- Hydration tracking
- Calorie tracking
- Daily hydration and calorie goals
- Automatic calorie calculation from drinks
- Custom drinks and foods
- Entry history
- Graph-based statistics and visualization
